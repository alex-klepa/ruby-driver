class Cassandra::TimeUuid::Generator

A UUID version 1, variant 1 generator. It can generate a sequence of UUIDs with reasonable uniqueness guarantees:

  • The clock ID and node ID components are set to random numbers when the generator is created.
  • If two calls to #next happen within the time afforded by the system clock resolution a counter is incremented and added to the time component.
  • If the clock moves backwards the clock ID is reset to a new random number.

Instances of this class are absolutely not threadsafe. You should never share instances between threads.

next

Returns a new UUID with a time component that is the current time.

Returns:
Type Details
TimeUuid a new UUID
Specifications:
Generator#next returns a UUID generated from the current time 
x = generator.next
x.to_time.to_i.should == 1370771820
x.to_time.usec.should == 329394
Generator#next returns unique IDs even when called within a time shorter than the clock resolution 
x1 = generator.next
x2 = generator.next
clock.stub(:usec).and_return(329394 + 1)
x3 = generator.next
x1.should_not == x2
x2.should_not == x3
Generator#next creates a pseudo random clock ID 
str = generator.next.to_s.split('-')[3]
str.should_not === '0000'
Generator#next uses the clock ID for all generated UUIDs 
str1 = generator.next.to_s.split('-')[3]
str2 = generator.next.to_s.split('-')[3]
str3 = generator.next.to_s.split('-')[3]
str1.should == str2
str2.should == str3
Generator#next creates a new clock ID when the clock has moved backwards 
str1 = generator.next.to_s.split('-')[3]
clock.stub(:to_i).and_return(1370771820 - 5)
str2 = generator.next.to_s.split('-')[3]
str1.should_not == str2
Generator#next creates a pseudo random node ID 
str = generator.next.to_s.split('-')[4]
str.should_not == '000000000000'
Generator#next uses the node ID for all generated UUIDs 
str1 = generator.next.to_s.split('-')[4]
str2 = generator.next.to_s.split('-')[4]
str3 = generator.next.to_s.split('-')[4]
str1.should == str2
str2.should == str3
Generator#next sets the multicast bit of the node ID (so that it does not conflict with valid MAC addresses) 
x = generator.next.value & 0x010000000000
x.should == 0x010000000000
Generator#next generates a version 1, variant 1 UUID 
x = generator.from_time(clock)
(x.value & 0x10008000000000000000).should == 0x10008000000000000000

from_time

(time, jitter = rand(2**16))

Returns a new UUID with a time component based on the specified Time. A piece of jitter is added to ensure that multiple calls with the same time do not generate the same UUID (if you want determinism you can set the second parameter to zero).

Parameters:
Name Type Details
time Time the time to encode into the UUID
jitter Integer (defaults to: rand(2**16)) a number of microseconds to add to the time to make it unique
Returns:
Type Details
TimeUuid a new UUID
Specifications:
Generator#from_time returns a UUID for the specified time with a bit of random jitter 
x = generator.from_time(clock)
x.to_time.to_i.should == 1370771820
x.to_time.usec.should be > 329394
Generator#from_time returns a UUID for the specified time with an offset 
x = generator.from_time(clock, 8)
x.to_time.to_i.should == 1370771820
x.to_time.usec.should == 329394 + 8
